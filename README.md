### ❓실행 방법

```
$ git clone https://github.com/HiGeuni/muflix
$ cd muflix
$ npm install
```

여기까지 진행을 한 후 src/config/api.js을 아래와 같이 만들어준다.

```javascript
export const api = {
	url:"http://3.37.109.116:4000",
};
```

다음 파일을 다운로드 받아서 src/ 아래에 넣어준다.

```
$ npm start
```

http://localhost:3000 ➡️ 홈페이지에 접속한다.
